##  Project Overview

This repository contains **20 hands-on practice problems** on Python's
`range()` function, progressing from **Beginner → Advanced level**.

It is designed for: - Students learning Python fundamentals - Interview
preparation - Coding practice drills - Strengthening loop & iteration
concepts

------------------------------------------------------------------------

## Topics Covered

-   Basic `range(stop)` usage
-   `range(start, stop)` usage
-   `range(start, stop, step)` usage
-   Reverse iteration
-   Even & Odd number generation
-   Multiplication tables
-   Factorial calculation
-   Prime numbers
-   Fibonacci series
-   Pattern printing
-   List comprehensions with `range()`
